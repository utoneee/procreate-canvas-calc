# Procreate 畫布換算

輸入目標尺寸，自動計算出適合在 Procreate 開的畫布設定，並顯示匯入 Photoshop 的步驟。

## 功能

- **目標尺寸輸入** — 支援印刷（300 DPI）與螢幕 / APP 兩種用途，單位可選 cm 或 px
- **作圖倍數** — 1× ～ 4×，建議以高倍數作圖再縮圖輸出
- **畫布設定輸出** — 自動換算畫布像素尺寸、最大圖層數、建議解析度
- **即時警示** — 超過 iPad mini 7 上限（3088 px）或圖層數偏少時標示警告
- **常用預設** — A4 印刷、A5 印刷、iPhone 螢幕、名片、方形貼紙
- **Photoshop 步驟** — 依用途顯示匯入 PS 的完整操作流程

## 使用方式

直接用瀏覽器開啟 `index.html`，無需伺服器或安裝任何套件。

## 說明

- 圖層上限以 iPad mini 7（Procreate 5X）為準
- 印刷模式預設 300 DPI，螢幕模式預設 72 DPI
- 僅供參考，實際圖層數可能因 Procreate 版本略有差異
